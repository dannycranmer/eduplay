For Parents & Teachers

This game introduces fractions through visual bar models — one of the most effective ways for children to build fraction sense. Three progressive modes:

  • Name the Fraction (Easy) — identify fractions from visual models using halves, thirds, quarters, and fifths
  • Compare Fractions (Medium) — determine which fraction is larger using side-by-side bar models
  • Equivalent Fractions (Hard) — find fractions that represent the same amount (e.g., 1/2 = 2/4 = 3/6)

Covers Common Core standards 3.NF.A.1, 3.NF.A.3, 4.NF.A.1, 4.NF.A.2. No data is saved or sent anywhere.
